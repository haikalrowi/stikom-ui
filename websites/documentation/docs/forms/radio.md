# Radio

`Radio` is used when only one choice may be selected in a series of options.

## Usage

```js
import { Radio } from "@stikom-ui/react";
```

```jsx live
<IFrame>
  <Radio>Radio</Radio>
</IFrame>
```

## Props

### `size`

```jsx live
<IFrame>
  <Stack direction="row">
    <Radio size="sm" name="radio">
      sm
    </Radio>
    <Radio size="md" name="radio">
      md
    </Radio>
    <Radio size="lg" name="radio">
      lg
    </Radio>
  </Stack>
</IFrame>
```
